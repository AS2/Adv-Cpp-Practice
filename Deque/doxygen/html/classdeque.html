Detailed Description

template<class elemType, class memoryAllocatorClass = memoryAllocator>
class deque< elemType, memoryAllocatorClass >

Double linked list class.

Double linked list class, which intitialized by element type and memory allocation class, which must include methods 'void * alloc(unsigned int size)' and 'void clear(void *ptr)'

Template Parameters
elemTypeelements type
memoryAllocatorClassmemory allocation strategy
